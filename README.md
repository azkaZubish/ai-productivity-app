# AI Productivity Planner 

An AI-powered productivity application that transforms long-term goals into structured roadmaps, milestones, and daily tasks using Large Language Models (LLMs).

Instead of manually planning how to achieve a goal, users simply describe what they want to accomplish, provide their available study/work hours, and the AI generates a personalized roadmap that adapts to their progress over time.

---

## Features

### MVP (Phase 1)

- User Authentication (JWT)
- Create Personal Goals
- AI-Generated Learning/Productivity Roadmaps
- Milestone Generation
- Daily Task Generation
- Dashboard with Goal Overview
- Task Completion Tracking
- Progress Visualization

# Getting Started

## Clone the Repository

```bash
git clone https://github.com/your-username/ai-productivity-planner.git

cd ai-productivity-planner
```

---

## Install Dependencies

### Root

```bash
npm install
```

## Start Development Servers

### Backend

```bash
npm run client
```

### Frontend

```bash
npm run server
```
